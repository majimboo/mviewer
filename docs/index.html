mviewer 2.0.0

Export Marmoset .mview scenes to glTF.

mviewer is a Rust-native exporter for static and animated Marmoset Viewer scenes. It reads .mview archives directly and writes glTF, texture assets, a runtime sidecar, and a generated browser viewer.

Download v2.0.0 View on GitHub

Release builds are produced for Windows x64, Linux x64, macOS arm64, and macOS x64.

What it exports

  • Static and animated .gltf scenes
  • Meshes, skins, animation, cameras, and lights
  • Textures plus merged alpha / metallic-roughness maps
  • mviewer.runtime.json and generated viewer.html

Install

  • Download the archive for your platform from Releases
  • Extract it anywhere
  • Run mviewer --help to verify the install

Quick start

mviewer input.mview output_dir

The exporter writes a glTF scene, binary buffer, textures, runtime sidecar, and browser viewer.
